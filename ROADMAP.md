# Roadmap

Här är vad vi planerar att arbeta på framöver. Ordningen kan förändras baserat på feedback och prioriteringar.

## 🎯 Nästa (Q3 2025)

### Klickbara tal och forkelhistorik
- [ ] Gör alla resultattal klickbara
- [ ] Visa "varför?"-modal när man klickar
- [ ] Visa stegvisa beräkningar
- [ ] Implementera "breadcrumb för matematik"
- [ ] Låt användaren klicka på varje steg för att se delen

**Varför:** Det är projektets signaturfunktion. Det gör appen både mer användbar och mer förtroendeingivande än befintliga kalkylatorer.

### Målläge (Goal Mode)
- [ ] Lägg till "Jag vill ha X kvar"-input
- [ ] Beräkna baklänges från målvärdet
- [ ] Visa max bostadspris
- [ ] Visa minsta försäljningspris
- [ ] Visa flexibilitetsalternativ

**Varför:** Det är hur människor faktiskt tänker när de handlar bostad.

### Förklaringsmotorn
- [ ] Lägg formler på alla resultat
- [ ] Visa antaganden tydligt
- [ ] Länka till källdokumentation
- [ ] Lägg till "Läs mer"-länkar

**Varför:** Det gör appen till ett utbildningsverktyg, inte bara en räknesnurra.

## 🔧 Senare (Q4 2025)

### Avancerad lånanalys
- [ ] Lånsamlingsöversikt
- [ ] Jämför bolånekostnader
- [ ] Visualisera amortering över tid
- [ ] Visa effektiv ränta

### Flerkalkylanalys
- [ ] Jämför flera scenarier sida vid sida
- [ ] Visa skillnader visuellt
- [ ] Export av jämförelse

### Integrationer
- [ ] Importera från Booli/Hemnet
- [ ] Spara till molnlagring (optional)
- [ ] Dela via Mail/SMS
